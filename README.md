# Search-Filter-Using-JavaScript
A simple and interactive employee search application built with HTML, CSS, and JavaScript. This project displays a list of employees and allows users to search employees by name using a starts-with search filter. The results update in real-time as the user types.

Features:

1.Live Search Filtering (real-time updates)

2.Search matches only the starting characters using startsWith()

3.Fully dynamic card rendering

4.Shows “No Data Found” message when no results match

5.Clean, optimized, and beginner-friendly code

6.Uses only Vanilla JS — no frameworks
